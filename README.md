# Simple Calculator App

A basic calculator built with HTML, CSS, and JavaScript that performs arithmetic operations.

---

## Features
- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Input validation
- Error handling (division by zero)

---

## Technologies Used
- HTML5
- CSS3
- JavaScript

---

## Purpose
Built to practice:
- DOM manipulation
- Functions
- Event handling
- Basic logic building
